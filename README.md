# QA Test Case Checklist (Beginner Python Project)

A simple command-line tool that allows manual QA testers to create,
track, and complete test steps. Built using only the concepts from
Day 1–26 of Angela Yu's 100 Days of Python.

## Features
- Add test steps
- Mark steps as completed
- View checklist
- Automatically saves to a text file

## How to Run
1. Install Python
2. Open a terminal in this folder
3. Run:

   python main.py

## QA Skills Demonstrated
- Understanding of test case structure
- Tracking test execution
- Organizing steps logically
- Basic scripting to support manual QA work
